# Cybersecurity Quiz

## Question 1
**Which cybersecurity weapon scans for use of default passwords, missing patches, open ports, misconfigurations, and active IP addresses?**

- vulnerability scanners
- packet analyzers
- packet sniffers
- password crackers

**Answer:** vulnerability scanners

---

## Question 2
**What describes a feature of credentialed scans?**

- They return fewer false positives and fewer false negatives.
- They are less invasive than non-credentialed scans.
- They do not require usernames and passwords to provide authorized access to a system.
- They try to exploit vulnerabilities and may even crash the target.

**Answer:** They return fewer false positives and fewer false negatives.

---

## Question 3
**Which approach provides automated tools allowing an organization to collect data about security threats from various sources?**

- SuperScan
- SOAR
- Nmap
- NETCAT

**Answer:** SOAR

## Question 4

## Question 5
**A new person has joined the security operations team for a manufacturing plant. What is a common scope of responsibility for this person?**

- managing redundancy operations for all systems

## Question 6
