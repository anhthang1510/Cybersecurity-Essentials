# 7.0 - Introduction

## Tổng quan

Module 7 giới thiệu hệ điều hành Windows, các khái niệm vận hành cơ bản và những công cụ dùng để quản trị, giám sát, bảo mật Windows endpoints. Đây là nền tảng quan trọng vì Windows xuất hiện rất phổ biến trong môi trường cá nhân, doanh nghiệp và hệ thống máy chủ.

## 7.0.1 Why Should I Take this Module?

Windows đã phát triển qua nhiều thế hệ, từ các phiên bản GUI chạy trên MS-DOS đến Windows NT hiện đại. Với cybersecurity analyst, Windows là môi trường cần hiểu rõ vì nhiều sự cố bảo mật, malware, persistence, credential theft và misconfiguration xảy ra trên endpoint Windows.

Học module này giúp bạn:

- Hiểu cách Windows hoạt động ở mức OS, không chỉ ở mức giao diện người dùng.
- Biết các công cụ quản trị Windows quan trọng.
- Nhận diện điểm yếu thường gặp trong cấu hình Windows.
- Có nền tảng để kiểm tra process, services, registry, logs và network connections khi điều tra sự cố.

## 7.0.2 What Will I Learn in this Module?

Trong module này, bạn sẽ học:

- Lịch sử và sự phát triển của Windows.
- Kiến trúc Windows và cách hệ điều hành vận hành.
- Công cụ cấu hình, giám sát và quản lý tài nguyên hệ thống.
- Các phương pháp giúp giữ Windows an toàn, gồm update, malware protection, firewall, local security policy và log analysis.

## Mục tiêu học tập

- Mô tả lịch sử Windows từ DOS/MS-DOS đến Windows NT.
- Giải thích user mode, kernel mode, HAL, kernel, drivers và file systems.
- Hiểu boot process, startup, shutdown, processes, threads, services và registry.
- Sử dụng công cụ quản trị như CMD, PowerShell, WMI, Task Manager, Resource Monitor và Event Viewer.
- Áp dụng nguyên tắc bảo mật Windows như least privilege, strong passwords, firewall, updates và Defender.

## Cần ghi nhớ

- Windows hiện đại dựa trên kiến trúc NT, không còn là DOS có GUI.
- Security analyst cần hiểu cả cấu trúc hệ thống lẫn công cụ quản trị.
- Registry, services, startup entries và scheduled/background processes là vị trí quan trọng khi điều tra malware.
- Windows Update, Defender, Firewall và Local Security Policy là các lớp bảo vệ cơ bản.
- Không nên dùng tài khoản Administrator cho công việc hằng ngày.
