# Quiz - Cybersecurity Essentials

## Question 1
**What is the function of the Cloud Security Alliance (CSA)?**

A. It produces a statement of applicability (SOA) which stipulates control objectives and audit controls to be implemented.  
B. It audits the CIA Triad objectives.  
C. It provides security guidance to any organization that uses cloud computing.  
D. It ensures total compliance with the ISO 27000 standards.

**Correct answer:** C. It provides security guidance to any organization that uses cloud computing.

---

## Question 2
**What do penetration tests and red team exercises achieve?**

A. They provide a list of malware that has successfully penetrated the firewall.  
B. They provide audit trails for all TCP connections in place at any given time.  
C. They provide audit controls for all NetBIOS connections made.  
D. They simulate attacks to gauge the security capabilities of an organization.

**Correct answer:** D. They simulate attacks to gauge the security capabilities of an organization.

---

## Question 3
**Which of the following principles is used by the U.S. government in its access control models?**

A. Job rotation  
B. Need to know  
C. Separation of duties  
D. Mandatory vacations

**Correct answer:** B. Need to know

---

## Question 4
**An organization has experienced several incidents involving employees downloading unauthorized software and using unauthorized websites and personal USB devices.**

**What measures could the organization implement to manage these threats?**  
**Choose three correct answers.**

A. Implement disciplinary action  
B. Provide security awareness training  
C. Run computers from a central server rather than localized hard drives  
D. Use content filtering  
E. Monitor all employee activity  
F. Disable USB access

**Correct answers:**
- B. Provide security awareness training
- D. Use content filtering
- F. Disable USB access

## Question 5
**Match the data governance role to the correct function.**

| Function | Correct role |
|---|---|
| Oversees an organization’s data protection strategy | Data protection officer |
| Determines the purposes and means of personal data processing | Data controller |
| Processes personal data on behalf of the data controller | Data processor |
| Implements the classification and security controls for data | Data custodian |
| Ensures compliance with policies and procedures | Data owner |

## Question 6
**What is the primary goal of IT security governance?**

A. To define a set of controls that an organization should implement  
B. To make decisions to mitigate risk  
C. To provide oversight to ensure that risks are adequately mitigated  
D. To provide a set of policies and procedures to manage sensitive data

**Correct answer:** C. To provide oversight to ensure that risks are adequately mitigated.
