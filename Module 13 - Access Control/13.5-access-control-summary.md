# 13.5 - Access Control Summary

## Tổng quan

Module 13 đã xây dựng nền tảng đầy đủ về kiểm soát truy cập — từ lý thuyết mô hình đến triển khai giao thức thực tế. Mọi thứ đều xoay quanh ba câu hỏi cốt lõi: Bạn là ai? Bạn được làm gì? Bạn đã làm gì?

## Tổng kết theo chủ đề

**Access Controls:**
- Ba loại: vật lý (hàng rào, camera), logic (ACL, firewall, mã hóa), hành chính (chính sách, đào tạo).
- AAA: Authentication (xác minh danh tính) + Authorization (xác định quyền) + Accounting (theo dõi hành động).
- Identification: định danh duy nhất cho mỗi người dùng.
- MFA: ít nhất hai trong ba yếu tố (kiến thức, sở hữu, sinh trắc).
- Federated identity: tiện lợi nhưng mở rộng phạm vi rủi ro.

**Access Control Concepts:**
- Zero Trust: "never trust, always verify" — ba trụ cột: Workforce, Workloads, Workplace.
- DAC: linh hoạt nhất, chủ sở hữu quyết định.
- MAC: nghiêm ngặt nhất, nhãn và clearance bắt buộc.
- RBAC: phổ biến nhất doanh nghiệp, quyền theo vai trò.
- ABAC: linh hoạt nhất, nhiều thuộc tính.
- Rule-based: bộ quy tắc cụ thể (firewall/router).
- NAC: tự động hóa kiểm tra tuân thủ thiết bị khi kết nối mạng.

**Account Management:**
- Loại tài khoản: user, service, guest, administrator.
- Privileged accounts: mục tiêu số một; quản lý nghiêm ngặt; MFA bắt buộc.
- File permissions: Full Control > Modify > R&X > Write > Read; Deny ghi đè tất cả.
- Windows policy: mật khẩu, lockout, audit.
- HMAC: hash + secret key cho xác thực thông điệp.
- Giao thức: PAP (yếu), CHAP (tốt hơn), RADIUS, TACACS+, Kerberos (mạnh nhất).

**AAA Usage and Operation:**
- Local AAA: đơn giản, quy mô nhỏ.
- Server-based AAA: tập trung, mở rộng, audit tốt.
- TACACS+: mã hóa hoàn toàn, phân quyền lệnh, TCP 49.
- RADIUS: chuẩn mở, kết hợp auth/authz, accounting toàn diện, UDP 1812/1813.
- Accounting logs: start/stop record; command logging; bằng chứng compliance.

## Mối liên hệ giữa các bài

```
Kiểm soát Vật lý + Logic + Hành chính
         ↓
   AAA Framework (Authentication → Authorization → Accounting)
         ↓
   Mô hình kiểm soát truy cập (DAC/MAC/RBAC/ABAC/Rule/TAC/Zero Trust)
         ↓
   Quản lý tài khoản (Least Privilege, Privileged Accounts, File Permissions)
         ↓
   Giao thức thực thi (RADIUS/TACACS+/Kerberos)
         ↓
   Logging và Audit (AAA Accounting Logs)
```
