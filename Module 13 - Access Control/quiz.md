# Quiz - Access Control and AAA

## Question 1
**Which three processes are examples of logical access controls? (Choose three.)**

- [x] firewalls to monitor traffic
- [ ] guards to monitor security screens
- [x] biometrics to validate physical characteristics
- [x] intrusion detection system (IDS) to watch for suspicious network activity
- [ ] swipe cards to allow access to a restricted area
- [ ] fences to protect the perimeter of a building

**Answer:** firewalls to monitor traffic; biometrics to validate physical characteristics; intrusion detection system (IDS) to watch for suspicious network activity

---

## Question 2
**Match the type of multifactor authentication with the description.**

| Category | Correct option |
|---|---|
| something you know | a password |
| something you are | a fingerprint scan |
| something you have | a security key fob |

**Answer:**  
- something you know → a password  
- something you are → a fingerprint scan  
- something you have → a security key fob

## Question 9
