# 8.0 Introduction: Linux Overview

## Tổng quan

Linux là một hệ điều hành open source được dùng rộng rãi trên server, máy trạm, thiết bị mạng, hệ thống cloud và nhiều lab bảo mật. Điểm mạnh của Linux là khả năng tùy biến cao, chạy nhanh, có nhiều công cụ dòng lệnh và phù hợp cho cả vai trò client lẫn server.

Trong CyberOps và SOC, Linux quan trọng vì nhiều công cụ phân tích log, giám sát mạng, packet capture, SIEM connector và forensic utility chạy tốt trên nền Linux. Người làm bảo mật không nhất thiết phải là quản trị viên Linux chuyên sâu ngay từ đầu, nhưng cần đủ nền tảng để đọc lệnh, hiểu file cấu hình, xem service và nhận biết hành vi bất thường trên host.

## Ý chính

* Linux là open source OS, thường xuất hiện trong server, cloud, lab bảo mật và thiết bị chuyên dụng.
* Linux có thể đóng vai trò client, server hoặc nền tảng vận hành công cụ SOC.
* Shell và CLI giúp kiểm tra hệ thống nhanh, chính xác và dễ tự động hóa.
* Kiến thức Linux hỗ trợ các nhiệm vụ như hardening, log review, service monitoring và incident response.
* Module này đi từ khái niệm cơ bản đến file system, GUI, package, process và bảo vệ Linux host.

## Thuật ngữ quan trọng

| Thuật ngữ | Giải thích ngắn |

## Ghi chú chi tiết

### Vì sao Linux phổ biến trong bảo mật

Linux được ưa chuộng trong môi trường bảo mật vì có tính minh bạch, ổn định, dễ tự động hóa và hỗ trợ nhiều công cụ chuyên dụng. Trên một máy Linux, analyst có thể dùng CLI để xem network interface, đọc log, kiểm tra service đang nghe cổng, lọc text và chạy script xử lý dữ liệu.

### Linux trong vai trò client và server

Một Linux host có thể là máy người dùng, web server, DNS server, file server, sensor IDS hoặc máy phân tích mã độc trong lab cô lập. Cùng một nền tảng có thể phục vụ nhiều vai trò khác nhau nhờ hệ thống package và cấu hình linh hoạt.

### Mục tiêu của module

Sau module này, bạn nên có thể mô tả được Linux kernel và distro, dùng shell ở mức cơ bản, giải thích service và port, đọc file permission, hiểu mount point, nhận biết GUI components, quản lý package, xem process và nêu các biện pháp bảo vệ endpoint Linux.

## Ví dụ liên quan

Một SOC analyst nhận cảnh báo về lưu lượng bất thường từ một Linux web server. Analyst cần đăng nhập, kiểm tra service đang chạy, xem log trong `/var/log`, xác định process lạ và đánh giá xem hệ thống có cần cô lập hay không. Những thao tác đó đều dựa trên kiến thức nền của module này.

## Cần ghi nhớ

Linux là kỹ năng nền cho nhiều vai trò an ninh mạng. Hãy ưu tiên hiểu khái niệm trước, sau đó luyện lệnh thường dùng để biến kiến thức thành thao tác thực tế.
